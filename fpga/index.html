ZedBoard FPGA Design
Created and synthesized various projects on the ZedBoard and Xilinx Zynq FPGA.
details
As part of Iowa State's CPRE 4880 course, Embedded Systems Design, we completed various projects using the ZedBoard, a development kit that uses the AMD Xilinx Zynq-7000 chip. These projects ranged from implementing a Nintendo Entertainment System emulation to remotely controlling a quadcopter drone. We designed the hardware components of each project using the Vivado Design Suite, and we wrote embedded application code in C using Vitis.

Languages: C, VHDL, MATLAB
Tools: Vivado Design Suite, Vitis SDK, Linux
Technologies: Avnet ZedBoard, AMD Xilinx Zynq-7000 SoC
